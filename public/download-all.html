⚽ Elsayad Scores - تحديث التطبيق

اضغط الزر لتحميل ملف ZIP يحتوي كل ملفات المشروع المحدّثة

📋 خطوات التحديث:

  1. اضغط الزر واحفظ ملف elsayad-update.zip
  2. فكّ الضغط في مجلد مشروعك على جهازك (استبدل الملفات الموجودة)
  3. شغّل: DATABASE_URL="رابط-Neon" npx drizzle-kit push
  4. شغّل: npm run build للتأكد
  5. ارفع: git add . && git commit -m "تحديث" && git push
  6. Vercel سينشر تلقائياً خلال دقيقة! 🎉

⚠️ مهم:

  • لا تحذف ملف .env الخاص بك (فيه رابط قاعدة البيانات)
  • تأكد من وجود node_modules (شغّل npm install لو لزم)
